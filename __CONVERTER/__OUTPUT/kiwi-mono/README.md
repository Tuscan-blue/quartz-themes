## Usage

For `LIGHT` version:

```scss
// In light.scss
@import "quartz-themes/themes/kiwi-mono/light";
```

For `DARK` version:

```scss
// In dark.scss
@import "quartz-themes/themes/kiwi-mono/dark";
```
