## Usage

For `LIGHT` version:

```scss
// In light.scss
@import "quartz-themes/themes/material-gruvbox/light";
```

For `DARK` version:

```scss
// In dark.scss
@import "quartz-themes/themes/material-gruvbox/dark";
```
