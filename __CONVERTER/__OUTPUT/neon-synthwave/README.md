## Usage

For `LIGHT` version:

```scss
// In light.scss
@import "quartz-themes/themes/neon-synthwave/light";
```

For `DARK` version:

```scss
// In dark.scss
@import "quartz-themes/themes/neon-synthwave/dark";
```
