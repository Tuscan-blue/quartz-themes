## Usage

For `LIGHT` version:

```scss
// In light.scss
@import "quartz-themes/themes/ayu-light-mirage/light";
```

For `DARK` version:

```scss
// In dark.scss
@import "quartz-themes/themes/ayu-light-mirage/dark";
```
