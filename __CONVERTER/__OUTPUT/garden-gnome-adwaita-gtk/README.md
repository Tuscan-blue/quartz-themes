## Usage

For `LIGHT` version:

```scss
// In light.scss
@import "quartz-themes/themes/garden-gnome-adwaita-gtk/light";
```

For `DARK` version:

```scss
// In dark.scss
@import "quartz-themes/themes/garden-gnome-adwaita-gtk/dark";
```
