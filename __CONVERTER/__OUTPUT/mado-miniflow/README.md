- **Type**: `LIGHT`/`DARK`
- **Flavors**: `FLAVOR1`/`FLAVOR2`
- **Variants**: `VARIANT1`/`VARIANT2`/`VARIANT3`

Description...

## Usage

For `LIGHT` version:

```scss
// In light.scss
@import "quartz-themes/themes/mado-miniflow/light";
```

For `DARK` version:

```scss
// In dark.scss
@import "quartz-themes/themes/mado-miniflow/dark";
```
